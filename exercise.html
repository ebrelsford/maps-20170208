Mapping with Carto

Part One

  1. Open data.cityofnewyork.us
  2. Search for motor vehicle collisions.
  3. Click View Data to preview the data. Don't download the entire file right now since it is very large.
  4. Search again, this time for borough boundaries.
  5. Preview the borough boundaries and download them.

Part Two

Create a Carto account

First, let's create an account with Carto. The New School has an agreement with Carto that can get you extra space if you need it—I recommend using it!

  1. Go to the New School Carto sign up page.
  2. Create a new account.

Download some data

You should already have the borough boundaries from the previous part. If not, get them now.

Now let's get the Inside Airbnb data:

  1. Go to insideairbnb.com.
  2. Click Get the Data.
  3. Scroll to New York City.
  4. Click listings.csv.

Make a map

  1. Open your Carto account.
  2. Click New Dataset and drag-and-drop the boroughs zipfile onto the page.
  3. Create another dataset, this time uploading the listings from Inside Airbnb.
  4. Create a map with both of your layers.

Part Three

Open mapping session

  1. Use your own data or some of the following data sources to make a map with Carto:
    • NYC Open Data, I recommend starting with the datasets, filtered to map files, but consider looking at others too
    • NYC Department of City Planning Open Data
    • US Government's open data portal
    • World Bank's data bank
